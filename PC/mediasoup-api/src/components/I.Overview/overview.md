## mediasoup 总览

顶级导出模块。

```js
const mediasoup = require("mediasoup");

// Or using destructuring assignment.
const {
  types,
  version,
  observer,
  createWorker,
  getSupportedRtpCapabilities,
  parseScalabilityMode
} = require("mediasoup");
```

总共包含三个属性与三个顶级方法。

### 属性
- types
- version
- observer

### 方法
- createWorker
- getSupportedRtpCapabilities
- parseScalabilityMode
