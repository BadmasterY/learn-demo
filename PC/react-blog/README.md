# react-personal-blog
react+koa博客

## 技术栈：
- React
- React-router
- Redux
- Redux-saga
- Antd
- Koa
- Mongodb
- Mongoose
- webpack
- babel

## 需要实现的内容：

### 前端部分：
- 首页单屏展示页
- 登陆
- 用户中心
- 个人简介页
- 后台管理页
- 公告(暂定用来查看每次的更新与bug修复)
- 弹幕(用来替代评论)
- 功能性模块(Message/进度/提示等)

### 后端部分：
- mongoose数据库操作
- 路由管理
- 权限管理
- 基本的增删改查

### 其他：
- 多端支持(PC/移动/微信小程序)
- RESTful
